# 🌱 Smart Agricultural Production Optimization Engine (OptiCrop)

A Machine Learning and Flask-based web application that recommends the most suitable crop based on soil nutrients and environmental conditions to improve agricultural productivity and resource efficiency.

---

## 📖 Project Overview

Agriculture plays a crucial role in food production, but selecting the right crop based on soil and climatic conditions remains a challenge for many farmers. The **Smart Agricultural Production Optimization Engine (OptiCrop)** is designed to address this problem using Machine Learning.

The application analyzes important agricultural parameters such as **Nitrogen (N), Phosphorous (P), Potassium (K), Temperature, Humidity, pH, and Rainfall** to predict the most suitable crop for cultivation. The system helps farmers make informed decisions, improve crop yield, optimize fertilizer usage, and promote sustainable farming practices.

## 📊 Input Parameters

The system accepts the following input values:

| Parameter       | Description                 |
| --------------- | --------------------------- |
| Nitrogen (N)    | Nitrogen content in soil    |
| Phosphorous (P) | Phosphorous content in soil |
| Potassium (K)   | Potassium content in soil   |
| Temperature     | Temperature in °C           |
| Humidity        | Relative humidity (%)       |
| pH              | Soil pH value               |
| Rainfall        | Rainfall (mm)               |
